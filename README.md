# Posto Review

Documentacao inicial do projeto Posto Review.

Este repositorio agora esta dividido em dois documentos principais:

- [Prototipo](C:\Users\luckj\Documents\projetos pessoais\posto-review\posto review\PROTOTIPO.md): explica a ideia do app, problema, publico, MVP e fluxo de uso;
- [Tecnologia](C:\Users\luckj\Documents\projetos pessoais\posto-review\posto review\README_TECNOLOGIA.md): organiza a base tecnica sugerida para tirar o projeto do papel.

## Resumo

O Posto Review e um aplicativo colaborativo para motoristas encontrarem, cadastrarem e avaliarem postos de combustivel com base em experiencias reais.

## Objetivo desta Documentacao

Separar claramente:

- a visao do produto;
- a trilha tecnica de implementacao.

Isso facilita tanto a apresentacao da ideia quanto a organizacao do desenvolvimento.
